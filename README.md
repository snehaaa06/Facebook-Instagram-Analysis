# Meta Ad Performance Dashboard 📊

An interactive Power BI dashboard analyzing Meta (Facebook & Instagram) 
ad campaign performance.

## Features
- KPI tracking: Impressions, Clicks, CTR, Conversion Rate, Engagement Rate
- Demographic breakdown by Gender and Age
- Geographic click distribution (World Map)
- Weekly and Hourly trend analysis
- Ad Type comparison: Stories vs Video
- Dynamic filters: Campaign, Platform, Target Interest

## Tools Used
- Power BI Desktop
- Data Source: Meta Ads Dataset
